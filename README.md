# 🧬 Sequence Alignment Toolkit

A Python toolkit implementing classical sequence alignment algorithms for DNA and protein sequences. The project demonstrates dynamic programming techniques used in bioinformatics, including global and local sequence alignment, customizable scoring systems, and alignment visualizations.

## ✨ Features

### Global Alignment
- Needleman–Wunsch algorithm
- Configurable scoring
- Gap penalties
- Alignment score
- Identity percentage

### Local Alignment
- Smith–Waterman algorithm
- Local sequence similarity

### Protein Alignment
- BLOSUM62 scoring
- PAM matrices

### Visualizations
- Alignment matrix
- Traceback path
- Heatmaps
- Alignment viewer

### Command Line Interface
- FASTA input
- DNA and protein support
- Export results
